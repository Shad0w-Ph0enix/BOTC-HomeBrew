# Hydra ![image](https://github.com/yoyosource/BOTC-HomeBrew/assets/171445109/7f5f1376-d426-451b-ae6e-0a9b4b5d0320)


**Summary**

The hydra is a demon-type character with an ability inspired by its mythologic counterpart.
The game starts with two hydras in play, however there are no minions in play.
If you don't kill both hydras on the same day (meaning one survives), in the next night, another hydra is created
